The Starting Strength program

This is the first phase of the program

Day A Day B
Squat 5 reps x 3 sets Squat 5 reps x 3 sets
Press/Bench Press 5 reps x 3 sets Press/Bench Press 5 reps x 3 sets
Deadlift 5 reps x 1 set Deadlift 5 reps x 1 set

On B days, squat and deadlift is performed at 70% of the working weight

The shoulder press and bench press are alternated between A and B days.

Please see the starting strength website for additional programming information.
